Landing Page with a nice animation implemented with HTML and CSS.

## Description

This project is a landing page that I developed some years ago. The landing page features a smooth CSS animation and a responsive design. It includes a hero section, a navigation bar, social links, and animated bubbles.

## Project Structure


- `images/`: Contains all the image assets used in the project.
- `index.html`: The main HTML file that structures the landing page.
- `README.md`: The documentation file for the project.
- `style.css`: The CSS file that styles the landing page and implements animations.

## Features

- Responsive design
- Smooth CSS animations
- Navigation bar with a logo and sign-up button
- Hero section with a welcome message and call-to-action button
- Sidebar with social and useful links
- Animated bubbles

## Usage

To view the landing page, open the `index.html` file in a web browser.

## CSS Animations

The CSS animations are defined in the `style.css` file. The key animation is the bubble animation, which makes the bubbles float upwards and disappear.

```css
@keyframes bubble {
    0% {
        transform: translateY(0);
        opacity: 0;
    }
    50% {
        transform: translateY(0);
        opacity: 1;
    }
    70% {
        transform: translateY(0);
        opacity: 1;
    }
    100% {
        transform: translateY(-80vh);
        opacity: 0;
    }
}
```
